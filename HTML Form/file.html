input File
    input type file di gunakan untuk meng-upload berkas file dari komputer kita 
    saat kita membuat input type file, maka agar form mengirim berkas file nya kita
    menggunakan method 'post' dan juga enctype menggunakan multi/form-data
    jika kita menggunakan method 'get' maka data yang dikirim berupa query parameter
        dan file itu tidak akan di kirim, karna file itu tidak bisa d simpan dalam query parameter
