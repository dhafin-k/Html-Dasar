Tag select
    berfungsi sebagai wadah untuk membuat menu pilihan,
    sedangkan tag option digunakan untuk mendefinisikan setiap opsi yang dapat dipilih dalam menu tersebut.
        Atribut multiple 
    
    pada tag select memungkinkan pengguna memilih lebih dari satu opsi dalam dropdown. 
    Tanpa batasan jumlah, pengguna dapat memilih sebanyak opsi yang tersedia.
        atribute size
    
    pada size semisal ada 5 opsi/data,lalu kita beri atribute size dengan nilai 3
    maka yag di tampilkan pada dropdownnya akan 3 opsi/data tetapi nanti akan ada
    sebuah scrollbar yang bisa di scroll,jadi dari data tadi yang kita tampilkan hanya beberapa saja
    
    adalah elemen dalam HTML yang digunakan untuk mengelompokkan beberapa elemen option di 
    dalam elemen select. membantu mengorganisir pilihan dalam dropdown list dengan memberikan label untuk setiap grup pilihan.
   
    select: Elemen utama untuk membuat dropdown list.
    optgroup: Digunakan untuk mengelompokkan beberapa elemen option.
    label: Atribut yang mendefinisikan nama grup yang ditampilkan sebagai header grup.
    option: Elemen pilihan yang bisa dipilih oleh pengguna.
