Input type range adalah input untuk number / angka namun ditentukan jangkauan minimal dan minimal nya 
    Ini mirip seperti input number, yang membedakan adalah, 
    bentuk komponen inputnya tidak seperti input text, melainkan dalam bentuk slider, 
    yang bisa di slide ke kiri untuk menurunkan nilai dan ke kanan untuk menaikkan nilai 
    Secara default, nilai range naik bertahap sebanyak +1 atau berkurang sebanyak-1, 
    namun jika kita ingin mengubah nilai naik dan turun nya, kita bisa tambahkan atribute step
    ini itu sebuah garis panjang yang di beri titik untuk memberi nilai
