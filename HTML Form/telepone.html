Input type tel bisa digunakan untuk input informasi telepon 
    Namun berbeda dengan input email atau number, aturan di input tel harus kita tentukan sendiri, 
    hal ini karena format telepon di tiap negara berbeda-beda 
    Kita bisa mengubah aturan format nomor telepon menggunakan atribut pattern, 
    dimana isinya harus berisi Regular Expression Materi Regular Expression 
    akan dipelajari di kelas JavaScript, namun disini kita bisa mencoba-coba menggunakan 
    website berikut: https://regexr.com/
