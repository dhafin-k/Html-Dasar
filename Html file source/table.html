        Table
    Tag table, digunakan untuk membuat tabel 
    Tag tr untuk membuat row/baris 
    Tag th untuk membuat kolom header table 
    Tag td untuk membuat kolom isi table 
    Tag caption untuk keterangan table 
    Tag thead, untuk grup header table 
    Tag tbody, untuk grup isi table Tag 
    tfoot, untuk grup footer table

        Colgroup
    Jika ingin memberi style pada kolom tabel tanpa mengaturnya satu per satu, 
    gunakan tag  untuk mengelompokkan kolom dan  untuk setiap kolomnya. 
    Untuk menerapkan style yang sama ke beberapa kolom sekaligus, gunakan atribut span pada .
    ~Intinya
    colgroup: Kelompokkan kolom.
         col: Tentukan style kolom.
        span: Terapkan style ke beberapa kolom sekaligus.

        Colspan dan Rowspan
    Saat kita membuat tabel, kadang kita ingin menggabungkan beberapa kolom 
    atau beberapa baris, sehingga terlihat menjadi satu kolom 
    Pada kasus itu, kita bisa menggunakan atribut colspan untuk menggabungkan kolom dalam satu baris yang sama 
    Atau menggunakan atribut rowspan untuk menggabungkan beberapa kolom di baris yang berbeda
    colspan untuk baris ke kiri kanan
    rowspan untuk baris ke bawah
