Picture atau Media Responsif Selain untuk menampilkan gambar menggunakan tag img, di HTML juga bisa menggunakan tag picture untuk menampilkan gambar Salah satu keuntungan menggunakan tag picture adalah, kita bisa menggunakan beberapa lokasi gambar, dan bisa diatur sesuai dengan ukuran layar misalnya. Jadi ketika di layar besar, kita bisa kirim gambar dengan ukuran besar, dan ketika di layar kecil, kita bisa kirim gambar yang ukuran kecil Picture Content Tag img adalah void element, sedangkan Tag picture bukan Di dalam tag picture, kita bisa menambahkan tag source yang berisi lokasi gambar, dan tag img sebagai default gambar ketika semua kondisi tag source tidak terpenuhi
